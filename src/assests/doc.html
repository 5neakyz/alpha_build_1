The ML Unit Multi Stager

The Stager is a crudely put together Python Application. To be able to configure multiple ML units at once.

For connecting to a COM port and staging units you will get the results of the outcome in the following red / green format:

Green - Success: Connection Established, Staged Successfully

Red - Failure: Cannot Establish Connection, Staging Failed

For the Stager to successfully establish a connection with an ML unit, it must be able to read the main menu.
So, a connection failure may not necessarily be an indication of a faulty connection or lack of power, but a unit behaving erratically

Baud rate is 115200
File Transfer Protocol is Xmodem1k

If you select to push either Firmware or Personality, The Stager will Erase Config before pushing anything.
If you are only pushing a BLE file it will Not Erase Config
This is to prevent cases were the unit locks up as firmware and personality are not compatible.

The exact file size and the exact number of bytes sent may not precisely match. But it is still a good indication of progress.
