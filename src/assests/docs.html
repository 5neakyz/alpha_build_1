Quick Fire Guide

Select the com ports of the devices you need, you can unselect by clicking them again

Then press the connect button - wait for all Coms to to be displayed under the connect button

  • Green - ML is alive and talking
  • Red - Cannot talk to ML

You need to unselect any that return RED and press the connect button again

Now you can select the option you whish to run on the ML

You then click the select firmware / personality can pick what you need. If you are only pushing firmware you only need to select firmware.

Once all the Ports are green the run command will now be active and you can press the button

You will see the "Running..." label above the button to indicate that it is indeed doing something

once the command is done below the button will return if the command was successful for each port.

  • Green being successful
  • Red being unsuccessful
